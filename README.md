# Coin 🪙

> Money, with purpose.

Coin is a personal finance operating system designed to help people manage money intentionally.

## Features (Planned)

- 💰 Income Tracking
- 💸 Expense Tracking
- 🐷 Piggy Bank
- 🎯 Goals & Sinking Funds
- 📈 Investment Portfolio
- 🔄 Auto Sweep
- 📊 Analytics Dashboard

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase (planned)
